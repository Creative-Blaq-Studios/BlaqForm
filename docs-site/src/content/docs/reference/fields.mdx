Every field widget requires a `name` (used for form registration) and a typed `controller`. All fields auto-register with the nearest `BfForm` ancestor.

## Field widgets

| Widget | Value Type | Key Props | Description |
|---|---|---|---|
| `BfTextField` | `String` | `obscureText`, `maxLines`, `minLines`, `keyboardType`, `textInputAction`, `inputFormatters`, `autofillHints`, `prefixIcon`, `suffixIcon`, `hintText`, `labelText`, `enabled`, `readOnly`, `decoration` | Standard text input. |
| `BfTextField.password` | `String` | Same as `BfTextField` (auto-sets `obscureText: true`, `maxLines: 1`) | Password text input. |
| `BfTextField.email` | `String` | Same as `BfTextField` (auto-sets `keyboardType: TextInputType.emailAddress`) | Email text input. |
| `BfDropdownField<T>` | `T` | `items`, `itemBuilder`, `hintText`, `labelText`, `decoration`, `enabled`, `isExpanded` | Dropdown selector. |
| `BfCheckboxField` | `bool` | `label`, `subtitle`, `enabled`, `tristate` | Single checkbox toggle. |
| `BfSwitchField` | `bool` | `label`, `subtitle`, `enabled` | Material switch toggle. |
| `BfCheckboxGroupField<T>` | `List<T>` | `options`, `labelBuilder`, `enabled` | Multi-select checkbox group. |
| `BfRadioGroupField<T>` | `T` | `options`, `labelBuilder`, `enabled` | Single-select radio group. |
| `BfDateField` | `DateTime` | `firstDate`, `lastDate`, `dateFormat`, `hintText`, `labelText`, `decoration`, `enabled` | Date picker dialog. |
| `BfDateRangeField` | `DateTimeRange` | `firstDate`, `lastDate`, `hintText`, `labelText`, `decoration`, `enabled` | Date range picker dialog. |
| `BfPhoneField` | `String` | `defaultCountryCode`, `countries`, `labelText`, `hintText`, `enabled` | Phone number with country code selector. |
| `BfCurrencyField` | `double` | `symbol`, `decimalPlaces`, `thousandSeparator`, `decimalSeparator`, `labelText`, `hintText`, `decoration`, `enabled` | Locale-aware currency input. |
| `BfOtpField` | `String` | `length`, `obscureText`, `boxWidth`, `boxHeight`, `spacing`, `autofocus`, `labelText`, `enabled` | Multi-box OTP/PIN input. |
| `BfAutocompleteField<T>` | `T` | `optionsBuilder`, `displayStringForOption`, `debounce`, `optionBuilder`, `labelText`, `hintText`, `decoration`, `enabled` | Async search autocomplete. |
| `BfRatingField` | `double` | `maxRating`, `allowHalfRating`, `size`, `color`, `unratedColor`, `labelText`, `enabled` | Star rating. |
| `BfChipSelectField<T>` | `List<T>` | `options`, `labelBuilder`, `avatarBuilder`, `wrap`, `spacing`, `runSpacing`, `labelText`, `enabled` | Chip-based multi-select. |
| `BfSignatureField` | `List<List<Offset>>` | `height`, `strokeColor`, `strokeWidth`, `backgroundColor`, `showClearButton`, `labelText`, `enabled` | Signature drawing pad. |
| `BfSliderField` | `double` | `min`, `max`, `divisions`, `showValue`, `activeColor`, `inactiveColor`, `labelText`, `enabled` | Numeric slider. |
| `BfRangeSliderField` | `RangeValues` | `min`, `max`, `divisions`, `showValues`, `activeColor`, `inactiveColor`, `labelText`, `enabled` | Numeric range slider. |

## Layout widgets

| Widget | Key Props | Description |
|---|---|---|
| `BfFormSection` | `title`, `description`, `collapsible`, `initiallyExpanded`, `padding`, `children` | Groups fields with optional title and collapse behavior. Spacing uses `BfFormTheme.fieldSpacing`. |
| `BfFormRow` | `children`, `flexes`, `spacing` | Arranges fields horizontally with configurable flex ratios. |
| `BfSubmitButton` | `onSubmit`, `label`, `child`, `loadingWidget`, `disableWhenInvalid`, `style` | Auto-disable/loading submit button. |

## Common props on all fields

Every field widget shares these properties:

| Prop | Type | Description |
|---|---|---|
| `name` | `String` | Key used for form registration and value maps. |
| `controller` | `BfFieldController<T>` | The typed controller that owns value and validation state. |
| `enabled` | `bool` | Whether the field accepts user interaction. Defaults to `true`. |

## Controller extensions

Extensions available on specific controller types:

| Extension | On | Method/Property | Description |
|---|---|---|---|
| `BfFieldControllerStringX` | `BfFieldController<String>` | `.trimmed` | Returns `value?.trim()`. |
| `BfFieldControllerStringX` | `BfFieldController<String>` | `.wordCount` | Returns number of words in the value. |
| `BfFieldControllerWidgetX` | `BfFieldController<T>` | `.watch(builder)` | Shorthand for `ListenableBuilder` that passes the controller. |
| `BfFieldControllerSignatureX` | `BfFieldController<List<List<Offset>>>` | `.toPng(width, height, ...)` | Renders strokes to PNG bytes. |
| `BfFieldControllerSignatureX` | `BfFieldController<List<List<Offset>>>` | `.toBase64(width, height, ...)` | Renders strokes to base64-encoded PNG string. |
