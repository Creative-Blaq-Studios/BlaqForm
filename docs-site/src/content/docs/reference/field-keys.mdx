BlaqForm provides predefined string constants for common field names, giving you IDE autocomplete and eliminating typos. They're purely optional — plain strings still work everywhere.

## Usage

```dart
import 'package:blaq_form/blaq_form.dart';

BfFormBuilder(
  fields: {
    BfUserKeys.email: BfFieldConfig.email(
      label: 'Email',
      validators: [Bf.required(), Bf.email()],
    ),
    BfUserKeys.password: BfFieldConfig.password(
      label: 'Password',
      validators: [Bf.required(), Bf.minLength(8)],
    ),
    BfUserKeys.firstName: BfFieldConfig.text(
      label: 'First Name',
      validators: [Bf.required()],
    ),
  },
  builder: (context, form) => Column(children: [
    form.email(BfUserKeys.email),
    form.password(BfUserKeys.password),
    form.text(BfUserKeys.firstName),
    form.submitButton('Sign Up'),
  ]),
)
```

Constants are just `static const String` values — they're fully interchangeable with plain strings:

```dart
// These are identical:
BfUserKeys.email      // 'email'
'email'               // 'email'
```
