---
title: Installation
description: How to add FormKit to your Flutter project.
---

## Add the dependency

```yaml
# pubspec.yaml
dependencies:
  formkit:
    path: ../formkit  # or from pub.dev when published
```

Then run:

```bash
flutter pub get
```

## Import

```dart
import 'package:formkit/formkit.dart';
```

That single import gives you everything: validators, controllers, field widgets, builder, wizard, indicators, persistence, and logging.

## Requirements

- Flutter SDK `>=1.17.0`
- Dart SDK `^3.11.4`
- **Zero external dependencies** — FormKit only depends on the Flutter SDK
