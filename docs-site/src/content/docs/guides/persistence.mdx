BlaqForm's persistence layer is defined by the `BfFormPersistence` abstract class. Implement it once for your storage backend and use the same API everywhere.

## BfFormPersistence interface

```dart
abstract class BfFormPersistence {
  Future<void> save(Map<String, dynamic> values);
  Future<Map<String, dynamic>?> load();
  Future<void> clear();

  // Convenience methods (implemented in the base class):
  Future<void> saveForm(BfFormController controller);
  Future<void> restoreForm(BfFormController controller);
}
```

| Method | Description |
|---|---|
| `save(values)` | Persist the given key-value map. |
| `load()` | Retrieve the persisted map, or `null` if nothing has been saved. |
| `clear()` | Remove all persisted data. |
| `saveForm(controller)` | Serializes all field values via `controller.toMap()` and calls `save()`. |
| `restoreForm(controller)` | Loads persisted values and sets each matching field's value. Silently skips unregistered keys. |

## BfMemoryPersistence

A built-in in-memory implementation for testing and short-lived draft state:

```dart
final persistence = BfMemoryPersistence();

// Save
await persistence.save({'email': 'user@example.com', 'name': 'Jane'});

// Load
final values = await persistence.load();
// => {'email': 'user@example.com', 'name': 'Jane'}

// Clear
await persistence.clear();
await persistence.load(); // => null
```

Data lives only as long as the `BfMemoryPersistence` instance.

## SharedPreferences implementation

Here is how to implement persistence backed by `SharedPreferences`:

```dart
import 'dart:convert';
import 'package:shared_preferences/shared_preferences.dart';
import 'package:blaq_form/blaq_form.dart';

class SharedPrefsPersistence extends BfFormPersistence {
  final String key;

  SharedPrefsPersistence({this.key = 'bf_form_draft'});

  @override
  Future<void> save(Map<String, dynamic> values) async {
    final prefs = await SharedPreferences.getInstance();
    await prefs.setString(key, jsonEncode(values));
  }

  @override
  Future<Map<String, dynamic>?> load() async {
    final prefs = await SharedPreferences.getInstance();
    final raw = prefs.getString(key);
    if (raw == null) return null;
    return Map<String, dynamic>.from(jsonDecode(raw) as Map);
  }

  @override
  Future<void> clear() async {
    final prefs = await SharedPreferences.getInstance();
    await prefs.remove(key);
  }
}
```

Usage:

```dart
final persistence = SharedPrefsPersistence(key: 'signup_draft');

// Save form state
await persistence.saveForm(formController);

// Restore on page load
await persistence.restoreForm(formController);

// Clear after successful submission
await persistence.clear();
```

## Auto-save pattern

Use a listener on the form controller to auto-save whenever a field changes:

```dart
class AutoSaveForm extends StatefulWidget {
  const AutoSaveForm({super.key});

  @override
  State<AutoSaveForm> createState() => _AutoSaveFormState();
}

class _AutoSaveFormState extends State<AutoSaveForm> {
  late final BfFormController _formController;
  final _persistence = SharedPrefsPersistence(key: 'auto_save_draft');

  @override
  void initState() {
    super.initState();
    _formController = BfFormController();
    _formController.addListener(_onFormChanged);

    // Restore any previously saved draft
    _persistence.restoreForm(_formController);
  }

  void _onFormChanged() {
    // Debounce this in production to avoid excessive writes
    _persistence.saveForm(_formController);
  }

  @override
  void dispose() {
    _formController.removeListener(_onFormChanged);
    _formController.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return BfForm(
      controller: _formController,
      child: Column(
        children: [
          // ... fields ...
          BfSubmitButton(
            label: 'Submit',
            onSubmit: (controller) async {
              await controller.submit((values) async {
                await api.submit(values);
                // Clear draft after successful submission
                await _persistence.clear();
              });
            },
          ),
        ],
      ),
    );
  }
}
```

## Combining with BfDirtyGuard

Persistence pairs naturally with `BfDirtyGuard`. Save the draft on every change, and warn the user before discarding it:

```dart
BfDirtyGuard(
  controller: formController,
  title: 'Discard draft?',
  message: 'Your progress has been saved. Leave anyway?',
  child: MyFormBody(),
)
```
