## Your first form

```dart
import 'package:flutter/material.dart';
import 'package:blaq_form/blaq_form.dart';

class LoginForm extends StatelessWidget {
  const LoginForm({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Login')),
      body: Padding(
        padding: const EdgeInsets.all(24),
        child: BfFormBuilder(
          fields: {
            'email': BfFieldConfig.email(
              label: 'Email',
              hint: 'you@example.com',
              validators: [Bf.required(), Bf.email()],
            ),
            'password': BfFieldConfig.password(
              label: 'Password',
              validators: [Bf.required(), Bf.minLength(8)],
            ),
          },
          onSubmit: (values) async {
            // values['email'] and values['password'] are here
            await Future.delayed(const Duration(seconds: 1));
            print('Logged in: ${values['email']}');
          },
          builder: (context, form) => Column(
            crossAxisAlignment: CrossAxisAlignment.stretch,
            children: [
              form.email('email'),
              const SizedBox(height: 16),
              form.password('password'),
              const SizedBox(height: 24),
              form.submitButton('Log In'),
            ],
          ),
        ),
      ),
    );
  }
}
```

That's it. No `StatefulWidget`, no `initState`, no `dispose`, no controller wiring.

## What just happened?

1. **`BfFormBuilder`** created an `BfFormController` and two `BfFieldController`s from your config
2. **`form.email('email')`** returned a pre-wired `BfTextField` with email keyboard and your validators
3. **`form.submitButton('Log In')`** returned a button that auto-disables when invalid and shows a spinner during submission
4. **`onSubmit`** only fires after all validators pass
5. When the widget is removed from the tree, everything is disposed automatically
